🎮 Pattern Decoder
A challenging visual puzzle game that tests your pattern recognition skills! Observe flashing sequences on a 5x5 grid and decode the hidden rules across multiple levels of increasing complexity.

🚀 Quick Start
Installation
bash
npm install
npm run dev
Visit http://localhost:3000 to play!

Build
bash
npm run build
npm run preview
🎮 How to Play
Observe: Watch the 5x5 grid flash for 10 seconds

Decode: Identify the hidden pattern rule

Select: Click squares that match the pattern

Check: Submit your solution and see results

Advance: Progress to more challenging levels

📋 Level Patterns
Level	Rule	Description
1	Even indices	Squares where index % 2 === 0
2	Diagonals	Main diagonal and anti-diagonal
3	Prime numbers	Squares with prime-numbered indices
4	Center cluster	Center square and its 4 neighbors
5	Modular Arithmetic	(row + col) % 3 === 0
✨ Features
🎨 Visual Experience
3 Beautiful Themes: Light, Dark, and Matrix with special effects

React 18 with TypeScript

Zero UI Dependencies - custom components only

🎯 Scoring
✅ Correct Selection: +10 points

❌ Wrong Selection: -5 points

🔍 Missed Pattern: -2 points

🏆 Perfect Round: Bonus opportunities

Toggle sounds using the speaker icon in the top-right corner. All sounds are generated programmatically - no external files required!
